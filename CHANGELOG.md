# Changelog

## v0.1.0

Initial research prototype.

### Features

- CoPolDB lookup by monomer name
- RDKit canonical SMILES matching
- Exact and fuzzy search workflows
- Local CoPolDB HTML caching
- Molecular-weight extraction from monomer pages
- Mayo-Lewis stochastic copolymerization simulator
- Optional stochastic termination model
- Mass-based Mn, Mw, and dispersity calculations
- CSV export of simulation results
- Molecular-weight and DP distribution plots
- GitHub-integrated project structure

### Current limitations

- Constant feed composition
- No monomer depletion
- No composition drift
- No chain transfer
- No initiator kinetics
- Not yet a full Gillespie SSA implementation
